How it's work ?

Make your Choice

First of all, You need to choose either you cover both enterance & exit in a single camera or separated for each.


In a single Camera

In this, if someone goes below the prediction line then it will be counted as entered person else (goes above the prediction line) counted as exited person. The total number of present people will be calculated by substracted exited person from entered person at any instance of time.

In Separated Cameras

In this. one camera is used to calculate number of entered person by going down from the prediction line and other one is used to count exited person. Both cameras result i.e, number of entered and exited person will be stored as global variables. Now, we substract total number of exited person from entered person to count total people present in the mall at any instance of time.


About US

Crowd Counting is a web-software to count total number of present people inside the mall. We developed this tool because it will be more easier for the higher authorities to maintain records of daily visited people. We tried to make this system as simple as possible so that user does not face any issues while counting.

You are also able to see daily records in the excel file.
